JWSystem

O que é JWSystem ?

Não é um software oficial desenvolvido pelas Testemunhas de Jeová. É um software web independente desenvolvido por uma testemunha de Jeová para gerenciar assuntos gerais da congregação.

Armazenamento dos dados

Os dados são armazenados em nuvem num dos servidores do Firebase, um serviço oferecido pelo Google para desenvolvimento de aplicativos e sites. A própria ferramenta de login do google, que inclui outros métodos de entrada é aplicada para permitir acesso aos dados.

Cada usuário só pode acessar os dados da sua congregação, com permissões cadastradas por um administrador, que será o primeiro usuário responsável por criar a conta da congregação.

Módulos

O Programa está organizado em módulos. Na versão de lançamento (1.0) em 26/11/2023 estão disponíveis os módulos: Publicadores, Relatórios e Reuniões. Cada módulo pode ser configurado com permissões de leitura e gravação das informações pelo usuário administrador.

O módulo Publicadores permite cadastrar e consultar os publicadores por grupo, atualmente com limite para 10 grupos.

O módulo Relatórios permite enviar os relatórios no novo modelo válido a partir de outubro/2023. As consultas podem ser feitas por grupo/mês/ano.

O módulo Reuniões permite enviar as assistências das reuniões e consutar por mês/ano.

Responsabilidade

O usuário que cria a conta da congregação e gerencia o acesso de outros usuários à mesma é responsável pelo tratamento dos dados armazenados.
